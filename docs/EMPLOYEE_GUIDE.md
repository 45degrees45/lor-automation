# Employee Guide — LOR Automation Sidebar

## How to Use

### 1. Open the Sidebar
In any Google Doc, go to:
**Extensions → LOR Automation → Open Sidebar**

### 2. Fill in the Details

| Field | What to enter |
|-------|--------------|
| LOR Type | Select EB-1A, EB-2 NIW, or O-1A |
| Customer Doc URL | Paste the link to the customer's Google Doc (their profile, achievements, bio) |
| Recommender Name | Full name of the person signing the letter |
| Recommender Title | Their job title |
| Recommender Org | Their organization/company |

### 3. Generate Draft
Click **Generate Letter**.
The draft will be inserted directly into your Google Doc (takes ~15–20 seconds).

### 4. Review the Draft
- Read through the generated letter
- The sidebar shows how many tokens were used and the cost (usually < $0.02)
- Make any manual edits needed

### 5. Send for Review
Share the doc with your team lead as usual.
**Important:** Ask them to use Google Docs **Comments** (not suggestions) for feedback — this is what the system learns from.

---

## Tips for Better Output

- **Customer doc should be detailed** — the more achievements, metrics, and context the customer provides, the better the letter
- **Specify the recommender's field** in the customer doc (e.g., "recommender is a Professor of Computer Science at Stanford")
- **Don't edit the draft before team lead sees it** — the system learns from the gap between draft and final, so early edits can confuse the feedback

---

## Understanding the Cost Display

The sidebar shows:
- Input tokens (customer doc + context)
- Output tokens (the generated letter)
- Estimated USD cost

This feeds into the team dashboard. No action needed from you.

---

## Troubleshooting

| Issue | Fix |
|-------|-----|
| "Failed to generate" error | Check that customer doc URL is shared (not private) |
| Draft looks generic | Customer doc lacks specific achievements — ask for more details |
| Sidebar not loading | Refresh the Google Doc and reopen the sidebar |
